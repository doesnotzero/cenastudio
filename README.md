## 📋 Sobre

Cena Studio é uma plataforma SaaS completa para gestão de produtoras de vídeo, desenvolvida do zero por nossa equipe.

O software integra:
- 🎬 Gestão de projetos e produção
- 📝 Ferramentas de briefing, roteiro e orçamento
- 🎥 Salas de review com comentários em vídeo
- 🤖 Geração de documentos com IA
- 💼 CRM e gestão comercial
- 📊 Analytics e relatórios
