## 🔑 Informações Rápidas

### Stack
```
React 19 + Vite 7 + TypeScript 5
TailwindCSS v4 + Radix UI
Express + Prisma + PostgreSQL (Supabase)
OpenRouter (IA)
Vercel (Deploy)
```

### Comandos Úteis
```bash
npm run dev          # Dev server
npm run test         # Testes (150 passando)
npm run build        # Build produção
npm run e2e          # E2E tests
npm run check        # Type check
```

- ✅ Testes: 150/150 passando (100%)
- ✅ Build: 34-36s
